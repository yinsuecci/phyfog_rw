# PhyFog Radiant War

光学辐射主题的多人实时对战游戏，附带地图编辑器。玩家在二维网格上操控光塔，发射不同波段的光线，利用反射、折射与建筑布局进行对抗。

在线试玩（部署实例）：[rw.udclass.top](https://rw.udclass.top)

## 功能概览

- **五种波段**：无线电、红外线、可见光、紫外光、γ 射线，各有穿透、伤害、视野等规则
- **地图元素**：墙壁、平面镜、透镜、太阳能板、铅板、进攻塔、信标
- **服务器权威联机**：Socket.io 房间制，6 位房间号匹配，逻辑在服务端运行
- **地图编辑器**：可视化编辑地图并导出 JSON，游戏内导入开局
- **移动端适配**：双指缩放/拖屏、触控发射、可见光能量条、右下角瞄准圆盘

## 环境要求

- [Node.js](https://nodejs.org/) 18+（推荐 LTS）
- 现代浏览器（Chrome / Edge / Safari / Firefox）

## 新手教程

首页右上角点击 **进入教程**，在本地沙盒中按步骤学习：

- **P1 / 友军 P2 / 敌军 P3** 三方布局
- **五种波段**：无线电 → 红外线 → 可见光（含能量调节）→ 紫外光 → γ 射线
- **四种建筑**：平面镜、透镜、太阳能板、进攻塔
- **信标占领**与 **γ 射线歼敌**

无需创建房间或联机。

## 本地运行

```bash
npm install
npm start
```

默认监听 `http://localhost:3000`：

| 地址 | 说明 |
|------|------|
| `/` | 游戏（创建/加入房间） |
| `/map-editor.html` | 地图编辑器 |

### 局域网联机

1. 房主在本机运行 `npm start`
2. 其他玩家浏览器打开 `http://<房主局域网IP>:3000`（例如 `http://192.168.1.5:3000`）
3. 房主：导入地图 → 创建房间 → 将 **6 位房间号** 发给其他人
4. 其他人：加入房间 → 输入昵称与房间号 → 准备 → 房主开始

客户端会自动连接当前站点，无需手动填写服务器地址。

### 公网联机

将项目部署到公网可访问的服务器（VPS、云主机等），所有玩家访问同一域名即可。部署说明见 [DEPLOY.md](DEPLOY.md)。

## 地图编辑器

1. 打开 `/map-editor.html`
2. 放置元素、调整参数（平面镜方向圆盘、透镜焦距等）
3. 导出 JSON
4. 在游戏首页「创建房间」时导入该文件

编辑器与游戏共用镜面几何（`mirrorUtils.js`），角度约定：**0° 向东，90° 向南**。

## 操作说明

### 桌面端

| 操作 | 功能 |
|------|------|
| ← → | 旋转光塔（5° 步进） |
| Space | 发射当前波段 |
| 长按右下角圆盘 | 精确瞄准（八方向锁定） |
| 侧栏建设 | 在视野内点击格子放置建筑 |
| 选「平面镜」 | 右下角圆盘调整镜面方向 |
| 选「透镜」 | 侧栏滑动条调整焦距 |

### 手机端

- 单指拖屏、双指缩放
- 侧栏或底部控件调整可见光能量与发射
- 右下角圆盘与桌面端相同

## 项目结构

```
├── server.js          # Express + Socket.io 入口
├── serverGame.mjs     # 服务端游戏逻辑封装
├── map-editor.html    # 地图编辑器（单页）
├── public/
│   ├── index.html     # 游戏客户端
│   ├── css/game.css
│   └── js/
│       ├── main.js        # 客户端 UI、输入、渲染循环
│       ├── GameLogic.js   # 游戏规则（客户端镜像 + 服务端共用）
│       ├── GameSync.js    # 联机状态软同步
│       ├── RayCaster.js   # 光路追踪
│       ├── MapRenderer.js # Canvas 渲染
│       ├── AimDial.js     # 瞄准/镜面方向圆盘
│       └── mirrorUtils.js # 平面镜线段几何
├── deploy.ps1         # 一键部署脚本
└── DEPLOY.md          # 部署到 rw.udclass.top 的详细步骤
```

## 部署

```powershell
# 复制配置模板后编辑 SSH 等信息
copy deploy.config.example.json deploy.config.json

# 一键部署（默认 direct 模式：SCP 直传服务器）
.\deploy.ps1
# 或双击 一键部署.bat
```

生成独立部署工具：`.\build-deploy-exe.ps1`
