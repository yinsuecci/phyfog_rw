# PhyFog Radiant War

光学辐射对战联机游戏 + 地图编辑器。

## 启动

```bash
npm install
npm start
```

浏览器打开 http://localhost:3000

## 如何加入房间

### 同一局域网（同一 WiFi）

1. **房主**运行 `npm start`，记下局域网 IP（如 `192.168.1.5`）
2. **房主**：创建房间 → 导入地图 → 获得 **6 位房间号**
3. **其他玩家**：打开 `http://192.168.1.5:3000`，服务器地址填同一地址
4. 加入房间 → 昵称 + 房间号 → 准备 → 房主开始

### 不同网络

所有人必须能访问**同一台 Socket.io 服务器**：

- **云服务器**：部署到 VPS，共用域名
- **ngrok**：房主运行 `ngrok http 3000`，分享 ngrok 地址
- **端口转发**：路由器转发 3000 端口到房主电脑

分享：**服务器地址 + 6 位房间号**

## 视野与视角

- 每玩家有 **二维视野数组** `visionGrid[y][x]`
- 屏幕中心锁定当前控制光塔，**不可拖屏**
- 视野扩大后相机自动拉远

## 操作

| 按键 | 功能 |
|------|------|
| ← → | 旋转光塔 |
| Space | 发射 |
